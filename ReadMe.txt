A Java application for analyzing and comparing Shakespearean plays, allowing:
- Textual analysis of individual plays
- Comparative analysis between two plays
- GUI interface with report generation

Features

Play Analysis
- Title extraction
- Character speech analysis (turns and word counts)
- Location identification
- Word frequency statistics
- Letter frequency analysis

Play Comparison
- Side-by-side character comparisons
- Unique word identification
- Jaccard Similarity scoring
- Top word comparisons
